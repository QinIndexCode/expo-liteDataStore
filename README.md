
# expo-litedataStore

**Lightweight, zero-config local storage for Expo apps**
**超轻量、零配置的 Expo 本地存储**



Simple JSON-based storage using `expo-file-system` + encrypted storage with `expo-secure-store`. No SQL, no native code — works in **Expo Go**.
